# Sends Guide

<Callout type="info">
  Sends lets you share tracks with other users and manage your shared content.
</Callout>

## Understanding Sends

The Sends feature allows you to:
- Share tracks with other users
- Track sharing history
- Manage shared content
- Monitor recipient engagement

## Sending Tracks

<Steps>
### How to Send Tracks

1. Select a track to share
2. Choose recipients:
   - Individual users
   - Groups
   - Email addresses
3. Add a message (optional)
4. Set sharing options
5. Send the track

### Sharing Options

Configure your send with:
- Download permissions
- Expiration date
- Privacy settings
- Collaboration rights
</Steps>

## Send Features

<Callout type="tip">
  Use advanced send features to enhance your sharing experience.
</Callout>

### Advanced Options

- **Batch Sending**: Send to multiple recipients
- **Scheduled Sends**: Set future send times
- **Templates**: Save common send settings
- **Auto-Expiry**: Set automatic expiration
