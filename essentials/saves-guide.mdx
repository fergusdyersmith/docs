# Saves Guide

<Callout type="info">
  Saves is your personal library of tracks you want to keep and access later.
</Callout>

## About Saves

The Saves feature allows you to:
- Build a personal track collection
- Bookmark tracks for later
- Create custom playlists
- Organize your favorite tracks

## Saving Tracks

<Steps>
### How to Save Tracks

You can save tracks from multiple sources:
1. From your Inbox
2. From shared links
3. From other users' profiles
4. From your own uploads

### Save Options

When saving a track, you can:
- Choose a destination folder
- Add custom tags
- Set privacy settings
- Add notes or comments
</Steps>

## Organization Features

<Callout type="tip">
  Keep your saves organized using folders, tags, and playlists.
</Callout>

### Organization Tools

- **Folders**: Group related tracks
- **Tags**: Add searchable labels
- **Playlists**: Create custom collections
- **Smart Lists**: Auto-organized by rules

## Playlist Creation

<Steps>
### Creating Playlists

1. Select tracks to include
2. Choose playlist settings:
   - Name
   - Description
   - Privacy level
   - Collaboration options
3. Arrange track order
4. Save and share

### Playlist Management

- Add/remove tracks
- Reorder tracks
- Set privacy
- Share with others
</Steps>

## Storage and Limits

<Callout type="warning">
  Saved tracks count towards your storage quota.
</Callout>

### Storage Management

- Monitor storage usage
- Remove unnecessary saves
- Archive older tracks
- Use smart cleanup tools
