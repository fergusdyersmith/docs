# Track Management Guide

<Callout type="info">
  This guide covers everything you need to know about managing your tracks on Trax.
</Callout>

## Upload Process

<Steps>
### Preparing Your Files

Before uploading, ensure your tracks meet these specifications:
- Supported formats: MP3, WAV, AAC
- Maximum file size: 500MB
- Recommended quality: 320kbps for MP3

### Uploading

1. Navigate to your dashboard
2. Click "Upload New Track"
3. Drag and drop or select your file
4. Wait for initial processing

### Adding Metadata

<Tabs items={['Basic Info', 'Advanced Info', 'Artwork']}>
  <Tab>
    Required fields:
    - Track title
    - Primary genre
    - Release type (Single/EP/Album)
  </Tab>
  <Tab>
    Optional fields:
    - BPM
    - Key
    - ISRC code
    - Copyright information
  </Tab>
  <Tab>
    Artwork requirements:
    - Minimum size: 1400x1400px
    - Maximum size: 3000x3000px
    - Format: JPG or PNG
  </Tab>
</Tabs>
</Steps>

## Track Organization

<Callout type="tip">
  Use playlists and tags to keep your tracks organized as your library grows.
</Callout>

### Playlist Management

- Create themed playlists
- Arrange tracks in custom order
- Set playlist privacy settings
- Share playlists with collaborators

### Tagging System

Use tags for:
- Genre classification
- Mood indicators
- Project status
- Release readiness
- Collaboration status

## Batch Operations

<Steps>
### Selecting Multiple Tracks

1. Hold Shift to select consecutive tracks
2. Hold Ctrl/Cmd for individual selections
3. Use "Select All" for entire library

### Available Actions

- Bulk delete
- Update metadata
- Change privacy settings
- Export track data
- Share with collaborators
</Steps>
